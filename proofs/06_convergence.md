<h1 align="center">Proof of Convergence</h1>

> Beliefs approach the truth as evidence accumulates.

---

## Theorem

**As the number of independent evidence observations increases, the posterior probability of the true root cause approaches 1, and the posterior probability of all false causes approaches 0.**

**Note on multiple root causes:** When an investigation has multiple true root causes (as the branching template supports), this theorem applies independently to each branch of the Why tree. Each branch has its own set of candidate causes. Convergence occurs per-branch: within each branch, the posterior concentrates on the true cause for that branch. The theorem does not assume a single global root cause.

---

## Setup

Given:
- r* is the true root cause
- P₀(r*) > 0 (the prior assigns nonzero probability to the true cause)
- Evidence observations e₁, e₂, ..., eₙ are gathered independently
- The likelihood model P(eᵢ | r*) is correctly specified

---

## Proof

**Step 1: Bayesian updating is multiplicative.**

After n evidence observations, the posterior is:

```
P(r* | e₁, e₂, ..., eₙ) = P(r*) × ∏ᵢ P(eᵢ | r*) / P(eᵢ)
```

For a false cause rⱼ ≠ r*, the posterior is:

```
P(rⱼ | e₁, e₂, ..., eₙ) = P(rⱼ) × ∏ᵢ P(eᵢ | rⱼ) / P(eᵢ)
```

**Step 2: The likelihood ratio diverges.**

The ratio of posteriors between the true cause and any false cause is:

```
P(r* | E) / P(rⱼ | E) = [P(r*) / P(rⱼ)] × ∏ᵢ [P(eᵢ | r*) / P(eᵢ | rⱼ)]
```

Since r* is the true cause, evidence is generated by r*. Therefore:

```
E[log P(eᵢ | r*) / P(eᵢ | rⱼ)] = KL(P(· | r*) ‖ P(· | rⱼ)) > 0
```

where KL is the Kullback-Leibler divergence. KL divergence is strictly positive when the two distributions differ (which they must, since r* ≠ rⱼ produce different evidence patterns).

**Step 3: The log-likelihood ratio grows without bound.**

```
log [P(r* | E) / P(rⱼ | E)] = log [P(r*) / P(rⱼ)] + Σᵢ log [P(eᵢ | r*) / P(eᵢ | rⱼ)]
```

By the law of large numbers, the sum grows linearly:

```
Σᵢ log [P(eᵢ | r*) / P(eᵢ | rⱼ)]  →  n × KL(P(· | r*) ‖ P(· | rⱼ))
```

Since KL > 0 and n → ∞, the log-likelihood ratio → ∞.

**Step 4: The posterior converges.**

As the log-likelihood ratio → ∞:

```
P(r* | E) / P(rⱼ | E) → ∞

Therefore:
  P(r* | E) → 1
  P(rⱼ | E) → 0  for all rⱼ ≠ r*
```

The posterior concentrates all probability mass on the true root cause. ∎

---

## Formal Name

This result is a special case of **Bayesian consistency for discrete hypothesis testing** (Doob, 1949). In a finite hypothesis space, the posterior concentrates on the true hypothesis with probability 1 as the number of independent observations grows, provided the true hypothesis has nonzero prior probability and the likelihood model is correctly specified.

**Note:** This is sometimes informally associated with the Bernstein-von Mises theorem, but BvM applies to continuous parametric models and concerns the asymptotic normality of the posterior — a different property. The correct theoretical foundation for HTSA's discrete hypothesis space is Doob's consistency theorem.

---

## Rate of Convergence

How fast does the posterior converge? The rate is governed by the KL divergence between the true and false hypotheses:

```
After n evidence observations:
  P(rⱼ | E) ≈ P(rⱼ) × exp(-n × KL(r* ‖ rⱼ))
```

- **Large KL divergence** → fast convergence. The true cause and false cause produce very different evidence. A few observations are enough.
- **Small KL divergence** → slow convergence. The true cause and false cause produce similar evidence. Many observations are needed to distinguish them.

In investigation terms: if two potential root causes would produce almost identical evidence, it takes more evidence to tell them apart. This is why the framework demands high-quality evidence at every node — weak evidence has small KL divergence and slows convergence.

---

## Required Conditions

The convergence guarantee requires:

| Condition | What Happens If Violated |
|:---:|---|
| P₀(r*) > 0 | If the true cause starts with zero prior, no amount of evidence can recover it. The posterior stays at zero. |
| Independent evidence | If evidence is correlated, the effective sample size is smaller than n. Convergence slows. |
| Correct likelihood model | If P(e \| r*) is misspecified, the posterior may converge to the wrong cause entirely. |

The most dangerous failure mode is **P₀(r*) = 0** — assigning zero probability to the true root cause before starting. This is why the framework insists on not naming a leading hypothesis before the 5 Ws are complete. Premature closure can set P₀ = 0 for the actual cause.
