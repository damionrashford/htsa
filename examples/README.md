## How to Read Them

Each example is a full investigation — all four layers, with annotations showing which math operates at each step.

Annotation key:

```
[GRAPH]      Graph theory (convergence, cycle, path)
[EXP]        Exponential problem space (branching factor, pruning)
[CAUSAL]     Causal inference (counterfactual, confounder)
[INFO]       Information theory (entropy drop, information gain)
[BAYES]      Bayesian reasoning (prior, update, posterior)
[SEARCH]     Search algorithm (DFS, BFS, Best-First)
[BIAS]       Cognitive bias warning
[EVIDENCE]   Evidence evaluation (tier, quality, conflict)
```

For the full reading guide, see **[00_index.md](00_index.md)**.
